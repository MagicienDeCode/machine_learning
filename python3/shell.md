# Shell

- **pwd** : current path, ou on est 
- **cd** .. : revernir sur le répertoire parent
- ls : affficher le content d'un répertoire
- cd [path]: entrer dans répertoire
- if python installed, python --version
- python [file]
- history : list commands déjà executé
- echo XX: afficher XX
- echo $HOME: afficher la valeur d'environment variable HOME
